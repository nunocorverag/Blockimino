Estas funciones son comunes en las bibliotecas de Arduino para controlar un LCD, como la biblioteca LiquidCrystal:

1.- La función lcd.cursor() se utiliza para mostrar el cursor en la pantalla LCD. Cuando esta función está activa, verás un guion bajo (_) en la posición actual del cursor en el LCD.

2.- La función lcd.noCursor() se utiliza para ocultar el cursor en la pantalla LCD. Después de llamar a esta función, el cursor ya no será visible.

3.- La función lcd.blink() se utiliza para hacer que el cursor parpadee en su posición actual en el LCD. El cursor se representará como un bloque que parpadea.

4.- La función lcd.noBlink() se utiliza para detener el parpadeo del cursor en el LCD. Si el cursor estaba parpadeando, dejará de hacerlo después de llamar a esta función.

5.- La función lcd.display() se utiliza para encender la pantalla del LCD. Después de llamar a esta función, los caracteres del LCD se mostrarán.

6.- La función lcd.noDisplay() se utiliza para apagar la pantalla del LCD. Los caracteres del LCD ya no serán visibles, pero permanecerán en memoria.

7.- La función lcd.scrollDisplayLeft() se utiliza para desplazar todo el contenido del display una posición a la izquierda. Esto puede ser útil si tienes más caracteres de los que puedes mostrar a la vez.

8.- La función lcd.scrollDisplayRight() hace lo contrario a scrollDisplayLeft: desplaza todo el contenido del display una posición a la derecha.

9.- La función lcd.autoscroll() se utiliza para cambiar automáticamente la dirección del texto que se está escribiendo en la pantalla. Cuando está activa, cada carácter nuevo se añadirá al principio, desplazando el resto del texto a la derecha.

10.- La función lcd.noAutoscroll() desactiva la función de autoscroll, por lo que los caracteres nuevos se añadirán al final del texto, que es la forma predeterminada.

11.- La función lcd.leftToRight() establece la dirección del texto de izquierda a derecha, que es la dirección más comúnmente usada.

12.- La función lcd.rightToLeft() establece la dirección del texto de derecha a izquierda. Esto puede ser útil en lenguajes que se escriben de derecha a izquierda, como el árabe.

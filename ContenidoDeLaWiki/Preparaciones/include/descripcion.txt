Este bloque representa la directiva #include. Esta directiva se utiliza para incluir una biblioteca en el código del proyecto. Las bibliotecas en Arduino contienen funciones que facilitan ciertas tareas, como controlar dispositivos específicos o realizar cálculos complejos. Al incluir una biblioteca, todas las funciones contenidas en esa biblioteca se hacen disponibles para usar en el código del proyecto.

El bloque tiene un solo campo que se llena con el nombre de la biblioteca que se quiere incluir en el proyecto. Puedes escribir el nombre de la biblioteca en el campo de texto dentro del bloque.

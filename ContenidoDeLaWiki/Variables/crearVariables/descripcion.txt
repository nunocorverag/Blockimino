Las variables en Arduino se utilizan para almacenar diferentes tipos de datos, como números enteros (integers), números decimales (floats), caracteres (chars), valores booleanos (true o false) y también pueden ser utilizadas para almacenar textos.

Como lo indica el nombre, este bloque nos permite crear variables de diferentes tipos y asignarles un nombre.

1.- Booleano (bool):
Valor mínimo: false (representa falso) Valor máximo: true (representa verdadero)

2.- Carácter (char):
Valor mínimo: -128 Valor máximo: 127
Representa un carácter individual, como 'a' o '9'.

3.- Cadena de caracteres (String):
No tiene un rango de valores fijo.
Representa una secuencia de caracteres, como "Hola" o "123".

4.- Número entero (int):
Valor mínimo: -32,768 Valor máximo: 32,767
Representa un número entero, como -100 o 500.

5.- Número largo (long):
Valor mínimo: -2,147,483,648 Valor máximo: 2,147,483,647
Representa un número entero más grande que el tipo "int".

6.- Número corto (short):
Valor mínimo: -32,768 Valor máximo: 32,767
Es similar al tipo "int", pero puede tener un rango más limitado en algunos microcontroladores.

7.- Número de punto flotante (float):
Valor mínimo: -3.4028235e+38 Valor máximo: 3.4028235e+38
Precisión: 6-7 dígitos decimales
Representa un número decimal con punto flotante, como 3.14 o -0.5.

8.- Número de punto flotante de doble precisión (double):
Valor mínimo: -1.7976931348623157e+308 Valor máximo: 1.7976931348623157e+308
Precisión: 15 dígitos decimales
Representa un número decimal con mayor precisión que el tipo "float".

1.- En el primer espacio el bloque permite que se le asigne un nombre a la nueva variable.
2.- El espacio de union en el borde derecho permite la asignacion de un valor, si no tiene ninguno por default sera cero.

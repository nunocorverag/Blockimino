El bucle for es una estructura de control que permite repetir un conjunto de instrucciones un número específico de veces. Esta instrucción consta de tres partes: inicialización, condición y incremento/decremento. En la inicialización se establece la variable de control (en este caso, "i"); la condición es la prueba que se realiza antes de cada iteración para determinar si el bucle debe continuar, y el incremento/decremento ajusta la variable de control después de cada iteración.

Este bloque de Blockly representa un bucle for en Arduino. El bloque tiene varios campos para configurar cómo funciona el bucle.

1.- int i =: Aquí puedes especificar el valor inicial de la variable i que será el contador del bucle.
2.- Operador de comparación: En este campo se puede seleccionar un operador que determinará la condición de parada del bucle (<, >, <=, >=, ==, !=).
3.- Entrada final: En esta sección, se especifica el valor con el que se comparará i para determinar cuándo se detiene el bucle.
4.- Dirección de incremento/decremento: Aquí puedes seleccionar si quieres que i incremente (++) o disminuya (--) en cada iteración del bucle.
5.- Bucle for: Este es el cuerpo del bucle donde puedes arrastrar y soltar otros bloques de código que se ejecutarán en cada iteración del bucle.
